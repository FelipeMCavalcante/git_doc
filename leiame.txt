Inicia o controle de Versionamento
git init

Adiciona o arquivo/alteração na estage area 
git add <arquivo>

Adicionando comentario do conteudo estage area
git commit -m "SEU_TEXTO"

Valida as alterações no projeto
git status

mostra o históric da "branch"
git log

ele cria uma nova "branch"
git checkout -b <nome_branch>
